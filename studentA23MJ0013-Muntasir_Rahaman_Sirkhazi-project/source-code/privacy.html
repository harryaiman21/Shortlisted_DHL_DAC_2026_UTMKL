Privacy Policy

This application allows users to sign in using their Google account in order to access files from their Google Drive. This access is used only to allow the user to select and import files they choose.

What data we collect

  • Basic profile information (such as name and email address)
  • Google Drive file metadata (file names, IDs, and types)

How we use the data

The data is used only to display and access the files you select from your Google Drive. We do not store, share, or sell your personal information or file contents.

Data storage

We do not store Google account data or Google Drive files on our servers. Access tokens are used temporarily during your session to retrieve file information.

Third-party services

This application uses Google APIs to authenticate users and access Google Drive. Google’s use of information received from Google APIs will adhere to the Google API Services User Data Policy.

Contact

If you have any questions about this Privacy Policy, please contact the developer.
